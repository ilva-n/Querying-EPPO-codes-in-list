# Querying-EPPO-codes-in-list
A tool to find EPPO codes for a list of botanical names, using database file eppocodes.sqlite which can be downloaded from EPPO data services. For cases when exact match is not found 
names in uploaded list are analyzed to try to guess reasons for non - matching and suggestions - similar bot names from database are searched.
You can see it working on Glitch https://find-eppo-codes.glitch.me/
